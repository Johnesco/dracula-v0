Dracula's Castle

v0 — Original BASIC

The original 1980s BASIC text adventure, playable in your browser via wwwBASIC (Google's JavaScript BASIC interpreter). You can also browse the source with syntax highlighting and annotations explaining the game engine, data structures, and puzzle logic.

Porting Notes

The original game was written for GW-BASIC. wwwBASIC compiles BASIC to JavaScript at load time and renders to a canvas using a CP437 bitmap font. Several compatibility fixes were needed:

  • Scalar/array name collision — GW-BASIC treats L (scalar) and L() (array) as separate variables; wwwBASIC shares one namespace, so assigning the room number to L corrupted the object-location array's memory. Patched wwwbasic.js to store array variables under a separate internal key.
  • NEXT X,Y — not supported; split to NEXT X:NEXT Y.
  • Implicit PRINT separators — GW-BASIC allows PRINT A$", "; wwwBASIC requires an explicit semicolon.
  • RUN 0 — not supported; replaced with GOTO 0.
  • PRINT newline clears to EOL — wwwBASIC fills the rest of the row with spaces; added trailing ; on lines that overlay the castle art.
  • TAB(n) produces tab chars — patched wwwbasic.js to emit spaces instead (tab glyphs render as white squares).
